Named types

Enum

enum Size {
    S,
    L,
    XL
};

Interface

interface IPerson {
    firstname: string;
    age?: number; //optional
    sayHello(): void;
    size?: Size;
}

Class

class Person extends IPerson {
    firstname: string;
    age: number;
    size: Size = Size.L;

    constructor() {}

    sayHello(): void {};
}

Object type literals

Object with implicit Any properties

{ foo: any; bar: any } or { foo; bar; }

Object with optional property

{ required: boolean; optional?: number; }

Hash map

{ [key: string]: Type; }

Indexable types

{ [id: number]: Type; }
